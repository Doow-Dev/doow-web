## Field naming

Fields are `snake_case` in the API and Title Case in the UI. The mapping
is deterministic — `vendor_name` becomes `Vendor name`, never
`vendor name` or `Vendor Name`.

## Reserved fields

A handful of fields are reserved and cannot be redefined: `workspace_id`,
`owner_id`, `created_at`, `archived_at`. Custom fields cannot collide
with reserved names.

## Custom fields

Workspaces can add custom fields against three resources: **app**,
**transaction**, and **vendor**. Custom fields are typed (`string`,
`number`, `enum`, `date`) and validated on write.
