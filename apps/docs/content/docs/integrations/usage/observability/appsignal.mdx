Connect AppSignal to pull performance metrics and usage data into Doow from your AppSignal organization.

## What you need

- An AppSignal account
- Your **App ID** for the application you want to connect
- A **Personal API token** from your AppSignal account

## Finding your App ID and API token

1. Go to [appsignal.com](https://appsignal.com) and open your organization
2. Navigate to the app you want to connect → **App Settings**
3. Your **App ID** is shown in the URL and app settings page — copy it
4. Navigate to **Profile → Personal API Tokens**
5. Click **Create token**, name it (for example, `Doow - Usage Read`)
6. Copy the token
7. Paste both the App ID and token into Doow when prompted

## What Doow reads

| Field | Description |
|---|---|
| Metric timeseries | Performance metric values per app and time window |
| App ID | The AppSignal app that generated the data |
| Date | The time window the usage occurred in |

Data is pulled from AppSignal's metrics API and aggregated into Doow's usage pipeline.

## What Doow does not read

- The content of any error events, exception messages, or stack traces
- User session data or request payloads
- Source code or environment configuration
- Billing payment methods or invoices

## How to connect

1. Go to **Company Settings → Integrations** in your Doow workspace
2. Find **AppSignal** and click **Connect**
3. A dialog opens asking for your App ID and Personal API token
4. Fill in both fields and click **Connect**
5. Doow verifies the credentials and begins the initial sync

## Confirm the sync worked

After connecting, open the integration detail page from **Company Settings → Integrations**. A healthy observability sync should show a connected state, a recent sync timestamp, and metric or event rows for the expected account, app, service, project, or environment.

If no metrics appear, verify that the metric exists in the observability platform for the selected time range and that the credential can read it.

## Troubleshooting

Use these checks when metrics are missing or incomplete:

- Confirm the API key can read the account, app, service, or project.
- Confirm the metric exists in the observability platform before connecting it to Doow.
- Confirm the metric has data in the selected time range.
- Check whether the provider aggregates or delays metric availability.
- Check the integration event log for credential, scope, or rate-limit errors.

## Disconnecting

Go to **Company Settings → Integrations**, find AppSignal, and click **Disconnect**. The stored credentials are deleted immediately. You can also revoke the token from your AppSignal profile under **Personal API Tokens**.

## Next steps

After metrics appear, connect identity or HRIS data if you need to attribute usage to users, teams, or departments.
