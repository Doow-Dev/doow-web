Use cloud platform integrations when usage data is exposed through cloud
monitoring, billing, account, subscription, or project data instead of a direct
provider usage endpoint.

## When to use this path

Choose this path when your AI usage runs through a cloud provider and the
provider records consumption at the cloud account, subscription, project,
region, deployment, or model level.

## What success looks like

After connecting a cloud platform, Doow should show usage rows for the expected
account, project, subscription, resource, region, deployment, or model. If the
connection succeeds but no usage appears, check that the credential can read
the cloud monitoring, billing, or usage data source for the right account.

## Next steps

Open the setup guide for your cloud provider. If you are still choosing a
usage-data path, return to [Choose a usage data source](/integrations/usage/choose-source).
