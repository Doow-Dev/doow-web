Start with the question you need Doow to answer. The best first integration is
the one that unlocks the answer you care about now.

<Callout title="If you are not sure" tone="success">
  Connect one usage data source first, then connect one identity provider. This
  gives Doow a practical baseline: what is being used and who is connected to
  that usage.
</Callout>

<Cards>
  <Card title="See usage" href="/integrations/usage/choose-source">

## Recommended paths

### Usage visibility

Use this path when you want to understand browser activity, desktop app
activity, AI provider usage, cloud platform usage, internal product usage, or
observability metrics.

<Steps>

1. Open [Choose a usage data source](/integrations/usage/choose-source).
2. Pick Doow Agent, direct provider APIs, cloud platforms, instrumentation,
   observability, or OTLP.
3. Connect the provider or method.
4. Confirm usage data appears in Doow.

</Steps>

### Ownership and user context

Use this path when you want to know who uses each app.

<Steps>

1. Connect [Google Workspace](/integrations/identity/google-workspace),
   [Microsoft 365](/integrations/identity/microsoft-365), or another identity
   provider.
2. Confirm users and app assignments sync.
3. Add HRIS if you need departments, managers, and employment status.

</Steps>

### Spend reconciliation

Use this path when finance needs a verified spend record.

<Steps>

1. Connect [Accounting integrations](/integrations/accounting).
2. Select the expense categories that contain software and SaaS spend.
3. Confirm expenses appear in Doow.
4. Add usage and identity data so Doow can explain whether spend matches use.

</Steps>
