Use this page to choose the source that gives Doow the data you need next. You
may be connecting Doow for the first time, adding another source later, or
checking what a source reads before asking an admin for access.

<Callout title="If you are unsure">
  Start with the question you want Doow to answer. Many teams begin with one
  usage source and one identity provider, but the right next source is the one
  that fills the data gap in front of you.
</Callout>

## Data layers

Each source family answers a different question. Connect the source family
that matches the question you need Doow to answer.

| Source family | Use it when | Start here |
|---|---|---|
| Usage data | You need consumption from apps, AI providers, cloud platforms, products, or metrics. | [Choose a usage data source](/integrations/usage/choose-source) |
| Organization data | You need users, groups, assignments, departments, managers, or employment status. | [Identity providers](/integrations/identity/google-workspace) or [HRIS](/integrations/hris) |
| Financial data | You need finance-confirmed spend, vendors, expense categories, or transaction dates. | [Accounting integrations](/integrations/accounting) |

## What success looks like

After connecting any source, confirm that Doow received usable data before
depending on it in reviews or workflows.

<Callout title="A healthy connection" tone="success">
  The integration shows a connected state, a recent sync or event timestamp,
  imported records in the detail page, and no credential or permission errors
  in the event log.
</Callout>

If the connection succeeds but no data appears, check the provider account,
credential scope, selected workspace, sync status, agent installation, and
tracking state before adding more integrations.
