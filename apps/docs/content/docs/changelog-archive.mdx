## Browse by month

The latest release notes live on the [Changelog](/changelog). This
archive collects older months so they stay searchable.

Each archived month carries the same structure as the latest entry: a
short summary at the top, a feature list, and a fix list. Anchors on
each month are stable, so external links stay valid.

## What gets archived

Anything older than ninety days from the current month rolls into the
archive automatically. We keep the most recent quarter on the live
changelog so subscribers see momentum without scrolling.
