## Approval shapes

Doow supports three approval shapes: **single approver** (default for low
amounts), **owner plus finance** (the common case), and **escalation
chains** (used above a configurable threshold). All three resolve to the
same audit trail.

## Routing rules

Routing is configured per-workspace, per-category. The most common rule
is *amount over X routes to finance*; everything else can stay with the
software owner.

## Auto-approvals

For repeating, low-risk spend, you can auto-approve based on a
combination of vendor, amount, and frequency. Auto-approvals still
record an entry in the audit trail.
