## What you'll do

This quickstart walks through creating a workspace, connecting one expense
source, and running a first spend review. The goal is to reach a working
review surface before you wire up the rest of your stack.

The full setup follows the same shape: connect, classify, review. The
quickstart skips classification rules so you can see results faster.

## Before you begin

You need workspace admin access and a corporate-card or expense source you
can read in test mode. A sandbox account is fine.
