## Why this matters

Most software contracts auto-renew. The negotiation window is the 60–90
days before renewal — miss it, and the vendor sets the next price. Doow
flags renewals as that window opens.

## How renewals are detected

Renewals come from three signals: contract end-dates you uploaded,
recurring transactions that match an existing vendor, and explicit
renewal flags from your AP system. Confidence scores are surfaced inline
so the reviewer can confirm or override.

## The renewal queue

The renewal queue is a single list ordered by negotiation deadline. Each
row links to the contract, the owner, and the spend trail.
