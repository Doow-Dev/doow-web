Doow sends notifications for sync events, spend threshold alerts, and product updates. You can configure which notifications you receive and how.

## Notification types

| Notification | Default | Description |
|---|---|---|
| Sync completed | On | Sent when an integration finishes its first sync |
| Sync failed | On | Sent when a sync fails after retries |
| Spend alert | Off | Triggered when spend exceeds a threshold you set |
| Weekly summary | On | Weekly digest of usage and spend highlights |
| Product updates | On | New features and improvements |

## Update your notification preferences

1. Go to **Settings → Notifications**.
2. Toggle each notification type on or off.
3. Changes save automatically.

## Set up a spend alert

Spend alerts notify you when monthly spend on a specific tool or category exceeds a threshold.

1. Go to **Settings → Notifications → Spend alerts**.
2. Click **Add alert**.
3. Choose the scope (workspace-wide, by vendor, or by team).
4. Enter a threshold amount.
5. Click **Save**.

You'll receive an alert via email and in-app when the threshold is crossed.

## Notification email address

Notifications are sent to your account email by default. To use a different address for a specific notification type (e.g., send spend alerts to a finance inbox):

1. Go to **Settings → Notifications**.
2. Click **Edit delivery** next to the notification type.
3. Enter the email address.
4. Save.
