You can export data from Doow at any time. Exports are available in CSV format from the dashboard, or programmatically via the API.

## Export from the dashboard

Most tables and charts in Doow have an **Export** button in the top-right corner.

1. Navigate to the report or table you want to export.
2. Click **Export → Download CSV**.
3. The file downloads to your browser immediately.

Exports include all visible rows, respecting any active filters.

## Bulk data export

For a full export of all workspace data:

1. Go to **Settings → Data management → Export**.
2. Select the data types to include (usage events, users, spend records).
3. Choose a date range.
4. Click **Request export**.

Doow prepares the export and emails a download link to the workspace owner within 30 minutes. Large exports may take longer.

## API access

The Doow API allows programmatic access to your workspace data. API keys are scoped to read-only by default.

1. Go to **Settings → API**.
2. Click **Generate API key**.
3. Copy the key and store it securely — it won't be shown again.

API documentation is available at [docs.doow.co](/docs).

## Data retention and deletion

Data collected by Doow is retained for the duration of your subscription plus 90 days. To delete specific data before retention expires:

1. Go to **Settings → Data management**.
2. Select the data type and date range.
3. Click **Delete selected data**.

This action is permanent and cannot be undone.
